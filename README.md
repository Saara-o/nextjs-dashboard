# Next.js App Router Course - Acme Dashboard
This project is the final outcome of the Next.js Dashboard learning module. The goal was to build a functional full stack dashboard application using modern Next.js features and best practices.

This project is part of Laurea's Full Stack Development course.

## Overview
The application includes authentication, a protected dashboard, navigation between pages, and a fully functional invoices section with CRUD operations. The customers page is included as a placeholder, as the module did not cover its full implementation.

## Features
- Next.js App Router with server components, layouts, nested routing, and page metadata

- Server Actions for form handling and database writes

- Zod for form validation and error handling

- PostgreSQL database managed through Vercel’s Neon instance

- Tailwind CSS for responsive UI styling

- NextAuth for authentication and route protection

- Invoices CRUD: list, filter, create, edit, delete

- Modern dashboard design with sidebar navigation, cards, icons, and clean layout
